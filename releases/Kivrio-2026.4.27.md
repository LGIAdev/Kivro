## What's Changed

- Removed the local OCR flow for image uploads sent to non-multimodal models while keeping file uploads for supported multimodal models.
- Migrated the embedded backend Python runtime from the legacy OCR branch to `runtime/backend-python`.
- Updated the Windows launcher and installer packaging to target the new embedded backend runtime path.
- Clarified the local architecture around the embedded scientific Python runtime in the project documentation.
