# Releases v2026.3.27.1

## Kivro 2026.3.27.1

[Full Changelog](https://github.com/LGIAdev/Kivro/compare/v2026.3.27...v2026.3.27.1)

- added local `pandas` support for Pyodide
- added local `seaborn` wheel support for Pyodide
- added on-demand loading for `pandas` and `seaborn`
- preserved the existing `Matplotlib` rendering pipeline
- validated basic `seaborn` chart rendering in Kivro
- documented the limitation of remote dataset loading such as `sns.load_dataset(...)`
